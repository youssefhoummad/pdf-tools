FAQ

Frequently Asked Questions

What is PyCompress?

PyCompress is a lossless image compression tool.

It can be used to reduce image sizes without significant quality loss. It is ideal for web content creators(Developers, bloggers e.t.c)

Is this available on my OS?

PyCompress is currently available for download and use on Ubuntu(Linux) and Windows. We'd be rolling out support for Mac OS soon.

How do I run the app after installation?

After downloading the app, navigate to the folder and locate the "PyCompress" executable, you can run it by just double-clicking on it. Alternatively, if you're familiar with the terminal, navigate to the root folder containing the downloaded app and run "./PyCompress". For Windows users, just run the downloaded executable.

How much does PyCompress cost?

PyCompress is totally FREE, but we're open to donations.
