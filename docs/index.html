FAQ

Frequently Asked Questions

What is PDF-tools?

PDF-tools is a collections of tools that work fine for pdf.

It can be used to split, crop, merge, convert to images, or from images to single pdf file.

Is this available on my OS?

PDF-tools is currently available for Windows user. We'd be rolling out support for Linux and Mac OS soon.

How do I run the app after installation?

After downloading the app, navigate to the folder and locate the "PDF-tools" executable, you can run it by just double-clicking on it. if you downloaded the zip file just decompress the zip file and double-clicking on PDF-tools.exe.

How much does PDF-tools cost?

PDF-tools is totally FREE, but we're open to donations.
